About This Project

This website is a project for the Introduction to CSS3 course from the University of Michigan. It demonstrates the ability to style multiple pages using a single external stylesheet.

Throughout this course, we learn how to manipulate layout, colors, fonts, and advanced positioning mechanics like Grid and Flexbox to make web pages responsive and accessible.
